# Poq

## How to run the project

1. Download source code
2. Make sure you have .NET 7 SDK
3. Open in JetBrains Rider or Visual Studio
4. Build
5. Hit F5 (Run)

## Tech Stack

- .NET Core
- MediatR
- FluentValidation
- AutoMapper
- Swagger
- Redis
- Polly
- xUnit
- Moq
- FluentAssertions
- WireMock.Net
